Four in a Square

  Four in a Square is a fast-paced two-player strategy game inspired by classic games like Connect Four and Reversi, with a unique sliding board mechanic.
  
  Players take turns placing pieces and sliding board tiles. The goal is to be the first to form a 2×2 square using four pieces of the same color. The rules are simple, but winning requires planning, spatial thinking, and strategy.

Game Info

  Players: 2
  
  Genre: Strategy
  
  Play Time: ~10 minutes
  
  Goal: Create a 2×2 square of your pieces before your opponent
